# Laboratório de JavaScript para QA

Este repositório é o meu laboratório de estudos para o **Módulo 2** do curso _Automação com Playwright_.

O foco aqui não é tornar-se desenvolvedor, mas aprender a "volante" da automação: saber conduzir, controlar e validar fluxos automatizados com confiança.

🎯 **Objetivo**

- **Aprender na prática** os fundamentos de JavaScript voltados para automação de testes com Playwright.
- **Praticar** exercícios pequenos e incrementais que possam ser reaplicados em cenários de QA.

📚 **Tópicos de Estudo**

- **Variáveis:** `let`, `const`
- **Tipos de Dados**
- **Arrays**
- **Funções**
- **Condicionais**
- **Loops**
- **Async/Await** (essencial para automação assíncrona)

🧠 **Mentalidade**

> Profissional não confia. Profissional confere.

💡 **Como usar este repositório**

- Clone o repositório e abra no seu editor preferido.
- Trabalhe nos exercícios na pasta correspondente (ex.: `exercicios/`) e crie commits pequenos e descritivos.

✨ **Dica rápida**

- Seja consistente: pratique diariamente, mesmo que por curtos períodos.
- Priorize entender o fluxo (o "volante") antes de otimizar código.
